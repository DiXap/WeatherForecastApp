# WeatherScratch
Project scratch from a WeatherApp

## You'll need
  - Python 3+ (LTS version)
  - pip installer:
    - jsonpickle
  
  - Python libraries:
    - json
    - requests
    - builtins
  
  - An [OpenWeatherAPIKey](https://openweathermap.org/)
    
    ```
    api = 'API_KEY'
    ```
    >`forecast.py`
